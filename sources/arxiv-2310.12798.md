The core problem addressed by MolCA is that large language models (LMs) typically represent molecules as 1D SMILES strings, inherently lacking the 2D graph perception required to comprehend topological structures. While prior cross-modal contrastive learning methods bridge molecular graphs and text, they are restricted to retrieval tasks and cannot support open-ended molecule-to-text generation, such as captioning or IUPAC name prediction. To resolve this, MolCA introduces a cross-modal projector and a uni-modal adapter, enabling an LM to perceive 2D molecular graphs as conditional inputs for generation.

The method follows a three-stage training pipeline. First, a five-layer GINE graph encoder (pretrained on ZINC15) extracts node features $f(g) = \mathbf{Z} \in \mathbb{R}^{|g| \times d}$ from a molecular graph $g$. A Q-Former acts as the cross-modal projector, utilizing learnable query tokens that interact with $\mathbf{Z}$ via cross-attention. In Stage 1, the encoder and projector are jointly optimized on molecule-text pairs using three objectives: Molecule-Text Contrasting (MTC), Molecule-Text Matching (MTM), and Molecule Captioning (MCap). Stage 2 connects the projector to a frozen LM (Galactica) and optimizes the cross-attention and encoder to generate text conditioned on both graph representations and 1D SMILES strings. Finally, Stage 3 fine-tunes the model for downstream tasks using a uni-modal LoRA adapter, updating weights via $\boldsymbol{h} = \mathbf{W}\boldsymbol{x} + \mathbf{B}\mathbf{A}\boldsymbol{x}$ while keeping $\mathbf{W}$ frozen.

Key optimization objectives include the MTC loss, which employs temperature-scaled cosine similarity: $\ell_{\mathrm{MTC}} = - \frac{1}{B} \sum_{i=1}^B \log \frac{\exp(\max_k \cos(\boldsymbol{m}_{ik}, \boldsymbol{t}_i)/\tau)}{\sum_{j=1}^B \exp(\max_k \cos(\boldsymbol{m}_{ik}, \boldsymbol{t}_j)/\tau)}$. The MTM loss trains a binary classifier on pooled query representations: $\ell_{\mathrm{MTM}} = \frac{1}{B} \mathbb{E}_{j,k \sim U(1,B)} \left[ \sum_{i=1}^B -\log \rho(g_i, \boldsymbol{y}_i) + \log \rho(g_i, \boldsymbol{y}_j) + \log \rho(g_k, \boldsymbol{y}_i) \right]$. Stage 2 minimizes the conditional language modeling loss: $-\log p_2(\boldsymbol{y} | \{\boldsymbol{m}_k\}_{k=1}^{N_q}, \boldsymbol{s}) = - \sum_{l=1}^L \log p_2(y_l | y_{1},..., y_{l-1}, \{\boldsymbol{m}_k\}, \boldsymbol{s})$.

Quantitatively, MolCA achieves state-of-the-art results across multiple benchmarks. On molecule captioning, MolCA with Galactica 1.3B (LoRA fine-tuned) attains 38.7 BLEU-2 on the PubChem324k dataset, surpassing baselines by 7.6 BLEU-2, and achieves 62.0 BLEU-2 on CheBI-20, outperforming baselines by 2.1 BLEU-2. For IUPAC name prediction on PubChem324k, it reaches 75.0 BLEU-2, exceeding baselines by 10.0 BLEU-2. In molecule-text retrieval, MolCA improves accuracy by over 20% on PubChem324k (66.6% M2T accuracy, 94.6% R@20) and achieves top performance on PCDes and MoMu datasets. Ablation studies confirm that combining 1D SMILES with 2D graph representations consistently outperforms using either modality alone, and the incorporation of 2D graphs significantly enhances the model's ability to count functional groups, reducing prediction error.

The authors acknowledge several limitations. The framework is strictly limited to generation tasks, excluding in-context learning and chain-of-thought reasoning. Current captioning performance remains insufficient for practical deployment, primarily due to the relatively small pretraining dataset scale (324k molecule-text pairs) compared to vision-language benchmarks (~10M pairs). Additionally, the model inherits standard LM risks, including potential generation of inaccurate or biased content, necessitating rigorous testing before real-world application.
