The core problem addressed is occluded person re-identification (re-ID), where traditional part-based methods rely on external semantic cues like pose estimation or human parsing. These external detectors are computationally expensive, prone to noise in cluttered environments, and frequently misalign in multi-pedestrian scenarios. To overcome this, the authors propose the Dynamic Patch-aware Enrichment Transformer (DPEFormer), an end-to-end architecture that automatically isolates discriminative body patches from occlusions without external supervision.

The method follows a structured pipeline. First, training images undergo Realistic Occlusion Augmentation (ROA), which synthesizes natural occlusions using masks generated by the Segment Anything Model (SAM). The augmented images are processed by a Vision Transformer (ViT) backbone, yielding a global classification token $f_g$ and $N$ patch tokens. Next, the Dynamic Patch Token Selection Module (DPSM) operates as a hard attention mechanism. It identifies a label-guided proxy token $f_{proxy}$ by finding the patch token with the highest similarity to $f_g$. Similarity scores between $f_{proxy}$ and all patches are computed, sorted in descending order, and their first-order gradients are calculated. The gradient with the maximum magnitude indicates a feature transition point, which serves as a dynamic splitting threshold $k$. A minimum threshold $k_{min}=48$ is enforced to prevent under-selection. The top $k$ tokens are retained and adaptively pooled into $P$ local part features. Subsequently, a Feature Blending Module (FBM) enriches these local features by embedding global context. Each part feature is split into 48 sub-parts, combined with learnable positional embeddings, and processed via a modified multi-head self-attention mechanism where queries derive from part features while keys and values derive from global features. The attended features are reshaped, added to the original part features, and passed through layer normalization and a feed-forward network. Finally, all blended part features are concatenated into a unified descriptor. Training utilizes a memory bank contrastive loss on pre-selection tokens and a cross-entropy identity loss on global, part, and final features.

Key mathematical formulations define the selection and blending processes. The proxy token is selected via $proxy = \arg \max_i (f_g f_1^T, f_g f_2^T, \dots, f_g f_N^T)$. Similarity scores $\mathbb{S}^p$ are sorted to compute gradients $\mathbb{D}[i] = \mathbb{S}^o[i] - \mathbb{S}^o[i+1]$, with the splitting point defined as $k = \arg \max_{i} (\text{Diff})$. The FBM employs a modified attention mechanism: $\text{Attention}(Q, K, V) = \text{Softmax}\left(\frac{Q K^T}{\sqrt{d_k}}\right)V$, where $d_k=16$. The final blended feature is computed as $f_{fbm_i} = \text{LN}(f_{part_i} + f_{mhsa_i}) + \text{FFN}(\text{LN}(f_{part_i} + f_{mhsa_i}))$, and the complete descriptor is $f_{final} = \text{Concat}(f_{fbm_1}, \dots, f_{fbm_P})$. The total loss combines contrastive and classification terms: $\mathcal{L}_{total} = \mathcal{L}_{con} + \mathcal{L}_{cls}$.

Quantitatively, DPEFormer achieves state-of-the-art performance across occluded and holistic benchmarks. On Occluded-DukeMTMC, it attains 69.9% Rank-1 accuracy and 58.9% mAP. On Occluded-REID, it reaches 87.0% Rank-1 and 79.5% mAP. For holistic datasets, it scores 95.4% Rank-1 and 88.1% mAP on Market-1501, and 90.0% Rank-1 and 80.3% mAP on DukeMTMC. Ablation studies confirm that ROA significantly outperforms Random Erasing and Cut&Paste, while the dynamic selection strategy with $k_{min}=48$ optimally balances feature retention and noise suppression.

The authors acknowledge several limitations. DPSM assumes a distinct feature boundary between body and occlusion/background tokens; if a pedestrian’s appearance closely matches occluding objects, the gradient-based split may incorrectly discard valid body tokens, resulting in insufficient feature coverage. Additionally, the ViT’s global self-attention mechanism exchanges information across all patches, causing feature representations to become complex and misaligned with the original image space, which occasionally leads to the selection of background or occluded patches. Furthermore, ROA is strictly an auxiliary training strategy and is not applied during inference. Finally, applying contrastive loss post-FBM degrades performance, indicating that body region information differs significantly between occluded and non-occluded images, making additional contrastive constraints detrimental to discriminative feature learning.
